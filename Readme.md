# MIND FEED
### A full-stack AI-powered personalised news discovery engine with semantic search using Qdrant and real-time news ingestion.
## TECH STACK
### FRONTEND
- React Js
- Tailwind CSS
- Shad CN
### BACKEND
- Node js
- Express js
- Postgres SQL
- Qdrant DB
- Hugging Face Transformer Model
- JWT
